# Patan — Claude Instructions

## Project
Single-player Catan clone with 2–4 NPCs. Built in Next.js (App Router) + TypeScript + Tailwind CSS. All board rendering is SVG-based (not Canvas).

## Dev Server
```
npm run dev   →   http://localhost:3000
```
